Publishing rule

What counts as a live page

  • The page must start from an official state source or statute.
  • The page must show a review date.
  • The page must state when a flat late fee is not clearly published.
  • The page must separate entity types when the state treats them differently.
  • The page must link back to the controlling filing page, portal, or statute.

Request another state

How to make a useful request

The best requests are narrow. Ask for a specific state, entity type, and filing rule. For example, "Texas franchise tax due date for LLCs" or "Florida annual report fee for domestic corporations" is much easier to verify than a general request to add many states at once.
