Montana annual report fee and deadline

Use this page if you need the Montana annual report deadline or current fee. Start with the April 15 deadline, then confirm whether the business is still active and in the current $35 late-filing lane or already needs reinstatement cleanup.

At a glance

Core due date Montana annual reports are due April 15.
Current filing fee $35 for active businesses filing after April 15, 2026.
Next waiver window Montana has separately announced another fee-waived filing window for reports filed between January 1 and April 15, 2027.

Current filing lane The active-business answer is now the $35 lane

Montana's fee table and help center point late but still-active businesses to the $35 annual report lane now that the April 15, 2026 waiver window has ended.

2027 planning The state has already announced another waived-fee window

Montana separately announced that it plans to waive the annual report filing fee again for businesses that file between January 1 and April 15, 2027.

Reinstating a business Inactive records usually require more than one payment

Montana domestic corporations and LLCs that need reinstatement may need to file each missing annual report at $35 and then complete the reinstatement filing.

Suspicious notice Use the official portal, not a look-alike solicitation

Montana warns that deceptive mailings and emails ask for annual report payments even though the official filing path runs through the Secretary of State portal.

Quick questions

What most filers need to know first

These answers keep the page short while still covering the questions people ask before they file.

Question Who should use this Montana guide?

Use this page for LLCs and corporations, including profit and nonprofit entities. Match the entity type on the official state record before you rely on the summary.

Question What due date should I start with for Montana?

Start with April 15 each year. If the state uses anniversary, report-month, or fiscal-year timing, the official state record still controls.

Question What fee or amount does this page show?

This page starts with $35 after April 15, 2026, with a separately announced waiver window for filings made from January 1 through April 15, 2027. If the amount changes by entity type, filing method, or status, confirm it on the official state source before paying.

Question What if I am already late?

$35 after April 15, plus reinstatement and delinquent-year charges if the entity is inactive. Confirm the current status and amount on the official state record before you pay or file.
