Arizona

Arizona annual report fee and deadline

Use this page if you need the Arizona corporation annual report due date, fee, or extension rule. Start by confirming the record is a corporation, because Arizona LLCs do not file annual reports with the Arizona Corporation Commission.

Recently checked Checked: April 10, 2026 Reviewed by FinLogic Hub Editorial Team 6 official sources Source: Arizona Corporation Commission

At a glance

Arizona LLC rule Arizona LLCs do not file annual reports. This rule applies to corporations.
Annual report fee $45 for for-profit corporations or $10 for nonprofit corporations.
Late path $9 per month for late for-profit annual reports, then pending inactive status and possible administrative dissolution.

Pick your case first

Start with the entity type that matches your record

Use the matching case below before you rely on a fee, filing path, or due date.

For-profit corporation Use the ACC due date and the $45 filing fee

Arizona for-profit corporations file each year by the designated ACC due date and face the published $9-per-month late penalty if the report is missed.

Nonprofit corporation Lower fee, but the filing is still annual

Arizona nonprofit corporations still file annual reports, but the fee is $10 and the FAQ says nonprofits are not assessed annual-report penalties.

Arizona LLC Stop before you use this page

Arizona LLCs do not file annual reports. If the record is an LLC, use the LLC-specific ACC record and notice path instead of this corporation guide.

Question What due date should I start with for Arizona?

Start with Each year by the designated due date shown on the ACC record, with a six-month extension available. If the state uses anniversary, report-month, or fiscal-year timing, the official state record still controls.

Question What fee or amount does this page show?

This page starts with $45 for for-profit corporations or $10 for nonprofit corporations. If the amount changes by entity type, filing method, or status, confirm it on the official state source before paying.

Question What if I am already late?

$9 per month for late for-profit reports; nonprofits are not assessed annual-report penalties, but missed filings still lead to delinquency notices and possible dissolution. Confirm the current status and amount on the official state record before you pay or file.
