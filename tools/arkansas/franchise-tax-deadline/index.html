Arkansas

Arkansas franchise tax deadline and annual report fees

Use this page if you need the Arkansas franchise tax deadline or the correct annual report form lane. Start with the entity type, because stock corporations, nonstock corporations, and LLCs do not use the same published amount.

At a glance

Main due date Due on or before May 1 each year.
Common fee lane $150 for stock corporations and LLCs, or $300 for nonstock corporations.
Late filing language Recent Arkansas SOS guidance says late franchise-tax filings are typically charged a $25 late fee plus 0.000274 percent interest per day.

Pick your case first

Start with the entity type that matches your record

Use the matching case below before you rely on a fee, filing path, or due date.

Stock corporation Start with the $150 minimum lane

Arkansas stock corporations use the capital-stock franchise-tax lane, with a $150 minimum tax shown on the official form set and forms page.

Nonstock corporation Use the separate $300 nonstock form lane

Arkansas nonstock corporations do not use the same minimum tax as stock corporations. The official nonstock form shows the $300 annual amount.

LLC or PLLC Use the Arkansas LLC franchise-tax lane

Arkansas LLCs and PLLCs owe the state franchise tax and use the separate LLC form lane instead of the corporate capital-stock calculation.

Quick questions

What most filers need to know first

These answers keep the page short while still covering the questions people ask before they file.

Question Who should use this Arkansas guide?

Use this page for Corporations, nonstock corporations, LLCs, banks, and insurance entities registered in Arkansas. Match the entity type on the official state record before you rely on the summary.

Question What due date should I start with for Arkansas?

Start with Due on or before May 1 each year. If the state uses anniversary, report-month, or fiscal-year timing, the official state record still controls.

Question What fee or amount does this page show?

This page starts with $150 for stock corporations and LLCs or $300 for nonstock corporations. If the amount changes by entity type, filing method, or status, confirm it on the official state source before paying.

Question What if I am already late?

Recent Arkansas SOS guidance says late filings are typically charged a $25 late fee plus 0.000274 percent daily interest. Confirm the current status and amount on the official state record before you pay or file.
